# Android Lab 06 - Componentes de UI Avanzados

Sexto laboratorio de Android. Implementacion de listas dinamicas con RecyclerView, adaptadores personalizados y patrones de navegacion entre pantallas.

## Temas cubiertos
- RecyclerView con adaptador personalizado
- ViewHolder pattern para rendimiento optimo
- Click listeners en items de lista
- Navegacion entre Activities/Fragments
- Material Design components
