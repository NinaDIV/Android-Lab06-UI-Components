# Laboratorio Android 06 — Componentes de UI Avanzados (RecyclerView y Navegación)

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=flat&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=flat&logo=android&logoColor=white)

Sexto laboratorio del curso de desarrollo Android. Implementación de listas dinámicas con RecyclerView, adaptadores personalizados y patrones de navegación entre pantallas, aplicando componentes de Material Design.

## 📚 Temas cubiertos

- RecyclerView con adaptador personalizado
- Patrón ViewHolder para un rendimiento óptimo
- Click listeners en ítems de lista
- Navegación entre Activities/Fragments
- Componentes de Material Design

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| Kotlin | Lenguaje principal del laboratorio |
| Android SDK | Plataforma de desarrollo |
| Material Design Components | Componentes de interfaz de usuario |
| Gradle | Sistema de compilación |

## ✅ Requisitos previos

Antes de ejecutar el proyecto asegúrate de tener instalado:

- [Android Studio](https://developer.android.com/studio) (versión reciente recomendada)
- JDK 17 (incluido con las versiones actuales de Android Studio)
- Android SDK con las herramientas de plataforma actualizadas (se instalan desde el SDK Manager)
- Un emulador de Android configurado en el AVD Manager **o** un dispositivo físico con la depuración USB habilitada

## 🚀 Instalación y ejecución

1. Clona el repositorio:
   ```bash
   git clone https://github.com/NinaDIV/Android-Lab06-UI-Components.git
   ```
2. Abre Android Studio y selecciona **Open**, apuntando a la carpeta del proyecto (`LAP05scaffold/`), que contiene el proyecto Android.
3. Espera a que Gradle sincronice las dependencias (se descargan automáticamente la primera vez).
4. Selecciona un emulador o conecta tu dispositivo físico.
5. Presiona **Run** (▶) o usa `Shift + F10`.
6. La aplicación se instalará y abrirá mostrando la lista dinámica construida con RecyclerView, desde la cual se puede navegar entre pantallas.
